# Keyboard Shortcuts

## Gmail

| Description          | Shortcut        |
| -------------------- | --------------- |
| Apply label          | V               |
| Remove label         | X               |
| Archive              | E               |
| Mute (for reply all) | M               |
| Go to Inbox          | G + I           |
| Reply                | R               |
| Reply All            | A               |
| Delete email         | Shift + 3       |
| Bullet list          | Cmd + Shift + 8 |
| Numbered list        | Cmd + Shift + 7 |
